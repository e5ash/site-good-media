3DS MAX

Разработка 3D графики с помощью наших графических дизайнеров для внедрения данных элементов в игры для мобильных устройств либо некоторые элементы самих приложений.

OBJECTIVE-C

Нами осуществляется доработка нативных мобильных приложений платформы iOS с написанных на Objective-C, а также осуществляем перевод приложений с Objective-C на Swift.

Данный язык программирования устаревает и разработка новых приложений осуществляется уже на языке программирования Swift.

PYTHON

Язык программирования, используемый в большей степени нами по пожеланиям заказчика либо при необходимости доработки каких-либо текущих проектов.

Так как обновления данного языка программирования происходит нечасто, поэтому по инициативе нашей компании к разработке на данном языке программирования мы прибегаем довольно редко.

MICROSOFT .NET

Язык программирования, нередко используемый для написания десктопных приложений семейства Windows.

ANDROID SDK

Универсальное средство, используемое нами, для разработки мобильных приложений Android OS.

Отличительной чертой от обычных редакторов для написания кодов является наличие широких функциональных возможностей, позволяющих запускать тестирование и отладку исходных кодов.

SWIFT

Разработка нативных мобильных приложений на базе iOS осуществляется на базе языка программирования Swift нашими специалистами.

На сегодняшний день данный язык программирования является наиболее актуальным и приспособленным для приложений на платформе iOS.

JAVASCRIPT

Наиболее распространенный язык программирования, часто используемый для написания сайтов и иных веб-программ. С его помощью можно более легко и точно придавать различную интерактивность страницам и прочим приложниям.

UNITY

Инструмент разработки, используемый для написания 2D и 3D приложений и игр.

С#

Технология используется для написания серверной части мобильных приложений, игр и БД, что позволяет выпускать продукты на рынок более функциональные и быстрореагирующие.

С++

Язык программирования общего назначения, в котором общее внимание уделено именно объектно-ориентированному программированию.

PHP

Язык программирования общего назначения, часто применяемый для разработки как веб-приложений так и для серверной части мобильных приложений.

CORELDRAW

Редактор в большей степени предназначенный для работы непосредственно с дополнительными продуктами заказчика (напр. разработка лого, отдельных элементов приложений либо сайта).

JAVA

Универсальный язык программирования, который может работать практически на любой компьютерной архитекторе и потому нередко используется для написания приложений.

NODE JS

Используется в некоторых случаях при написании серверной части (web-сервера) мобильных приложений, а также и десктопных приложений для более удобного взаимодействия с внешними сервисами.

PHOTOSHOP
Популярный редактор, семейства Adobe, используемый нами, для прототипирования и дизайна некоторых мобильных приложений.
SKETCH

При разработке дизайна мобильных приложений часто используется нашими специалистами данный редактор, который позволяет более глубоко проработать интерфейс и дизайн приложения.
